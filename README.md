## SZP博客源码

### 介绍

本博客采用[hexo](https://hexo.io)搭建,主题采用[material](https://github.com/viosey/hexo-theme-material)

> 博客地址:[SZP博客](https://blog.szpinc.org)

### 使用
1. 安装**nodejs**,
>[http://nodejs.cn/download/](http://nodejs.cn/download/)
2. 安装**hexo**
>npm install hexo-cli -g
3. 运行
>hexo s

### 注意
**运行之前需要安装依赖模块**
